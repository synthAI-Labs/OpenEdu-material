# Variable Modules in JavaScript

In JavaScript, variable modules are a common pattern used to encapsulate variables and create modular code. This approach helps in organizing and structuring your codebase.

## Example Module

Let's create a simple example of a variable module:

```js
// module.js

// Private variables
let privateVar = "I am private";

// Public variables
const publicVar = "I am public";

// Public function accessing private variable
function getPrivateVar() {
  return privateVar;
}

// Exporting public variables and functions
export { publicVar, getPrivateVar };
```

In this example, `privateVar` is a private variable, and `publicVar` is a public variable. The `getPrivateVar` function is a public function that can be used to access the private variable.

## Usage

To use the module in another file, you can import it like this:

```js
// main.js

import { publicVar, getPrivateVar } from "./module";

console.log(publicVar); // Output: I am public
console.log(getPrivateVar()); // Output: I am private
```

In this example, we import `publicVar` and `getPrivateVar` from the `module.js` file and use them in the `main.js` file.

By using variable modules, you can create reusable and maintainable code by encapsulating variables and exposing only what is necessary externally.
